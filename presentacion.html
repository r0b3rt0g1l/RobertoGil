La infraestructura

Un sistema profesional.
Sin servidores. Sin complicaciones.

GitHub

El código del sitio se guarda y versiona aquí. Todo el historial de cambios queda registrado. El repositorio es propiedad del municipio.

GitHub Pages

Publica el sitio al mundo automáticamente. Sin servidor propio. Sin mensualidad de hosting. Infraestructura de Microsoft — disponible 24/7.

Hostinger

Administración del dominio .com mientras llega el .gob.mx oficial. Correo institucional incluido.

Cloudflare

Seguridad, protección contra ataques, velocidad global y gestión de DNS. Activado desde el arranque.

Google Analytics

Monitoreo de visitas en tiempo real. Reporte mensual entregado al municipio.
